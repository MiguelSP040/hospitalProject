### hospitalProject ###

-- Dependencias 
Spring Data JPA 
Spring Web 
MySQL Driver

-- Procedimiento 
1.- Hacer una carpeta en su escritorio con un nombre distintivo 
2.- Dar clic derecho sobre la carpeta y seleccionar "Open Git Bash here" o parecido 
3.- En la terminal ingresar el comando "git clone" seguido del link del repositorio "https://github.com/MiguelSP040/hospitalProject.git" 
4.- Cerrar la terminal y repetir el paso 2 sobre la nueva carpeta que se creará dentro de la carpeta que seleccionaron o ingresar el comando "cd /nombre de la carpeta" *.- Esta nueva carpeta fungirá como el directorio raíz del proyecto, dicho de otra manera, será el punto de partida de nuestra estructura de archivo 
5.- Una vez dentro de dicha carpeta y con la terminal de Git Bash abierta, crear una rama con su nombre utilizando el comando "git branch", seguido de su nombre en MÍNUSCULAS 
6.- Verificar que estás en tu rama antes de realizar cualquier cambio 7.- En caso de no estar en tu rama, emplear el comando "git checkout", seguido de su nombre
